What is StatTrack


StatTrack is a tool built to allow users to go about tracking aspects of their life and then be presented with easy to read and appropriately represented data.

The project incorporates the use of an Android app which allows the user to input the data. Here we collect information such as who is inputting the data, where they are and the time. Using this data we can then work out relevant statistics, plot maps and create graphs.

This website acts as the web interface for the users and allows them to access the necessary content. Due to the design all of the content on the website updates in real time, including the graphs and maps.

StatTrack incorporates a number of different features such as real time databases, web authentication, web functions and handling a number of different API's. The project was designed as a learning experience.
